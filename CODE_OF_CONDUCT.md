# Code of Conduct

portaBaltica follows the [Contributor Covenant, v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short:

- **Be kind.** Critique ideas, not people.
- **Assume good intent.** Most misunderstandings are not malicious.
- **No harassment.** Of anyone. Anywhere associated with this project.
- **Respect the maintainer's call** on scope and direction — this is a
  research project, not a community-governed standard.

Report violations to the maintainer privately via the contact on
[naurolabs.com](https://naurolabs.com).
