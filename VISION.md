# portaBaltica — Baltic Open Data Intelligence

## What is this?

portaBaltica is NauroLabs' Moonshot v1 — the first project designed from day one to be self-sustaining. A public dashboard that aggregates Latvia's open data APIs into a single live intelligence view, with AI-powered insights.

Think Bloomberg Terminal for Baltic economies: real-time electricity prices, business registry changes, construction activity, weather, exchange rates — all in one place, interpreted by AI.

## Who is it for?

- **International companies** with Baltic operations (Nordic banks, real estate investors, consulting firms) — replaces 2-5 analyst-hours/week of manual data gathering
- **Journalists & researchers** — anomaly alerts and historical trends across 40+ government datasets
- **Developers** — unified REST API instead of integrating 52 different data sources
- **Latvian SME owners** — morning dashboard: exchange rates, electricity price, regulatory changes

## How does it connect to NauroLabs?

portaBaltica is the proof that an AI agent can generate revenue autonomously. The Baltic open data layer (9,000+ datasets across LV, EE, LT) is the connective tissue binding all NauroLabs experiments. Today, each project (agentMode, ERA, amberRepublic) hits government APIs independently. Tomorrow, portaBaltica becomes the shared data backbone — one API to rule them all.

## Revenue hypothesis

| Tier | Price | Value |
|------|-------|-------|
| Free | €0 | Public dashboard, today's data |
| Pro | €15/month | Historical trends, alerts, CSV export |
| Enterprise | €49/month | REST API, webhooks, white-label embed |

Break-even: 1 Enterprise or 3-4 Pro subscribers. Infrastructure cost: ~€5-18/month.

## Self-sustaining progression

1. **Prove €1** — an agent CAN earn money
2. **Cover costs** — ~€15-20/month infrastructure
3. **Fund experiments** — ~€50-100/month to finance new NauroLabs projects
4. **Real revenue** — €500+/month, replicable pattern

## Success metric

100 monthly active users on the free dashboard within 60 days of launch.

## Phased rollout

- **Phase 1:** 15 high-value Latvia APIs (economy, property, environment, maritime)
- **Phase 2:** 12 more APIs (business intelligence, transport, government)
- **Phase 3:** 10 more APIs (geospatial, maritime expansion, self-management agent)
- **Phase 4:** Baltic expansion — Estonia (40 APIs) + Lithuania (38 APIs)

## Design principles

1. **Data is free, intelligence is the product** — raw government data is CC0; our AI interpretation layer is the premium
2. **Dashboard-first** — not a newsletter, not an API. A live public dashboard that people visit daily
3. **Self-sustaining by design** — the agent tracks its own costs vs revenue and optimizes accordingly
4. **Standalone first, backbone later** — prove value independently, then become the shared data layer for all NauroLabs projects
